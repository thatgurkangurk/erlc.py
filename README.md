# erlc.py

a (better and open-source) python wrapper for the ERLC api.

current feature list:
- [X] raw get requests
- [X] raw post requests
- [ ] command sending
- [ ] higher level user management (built-in functions for kicking,banning etc)
- [ ] queue getter
- [ ] show server information
- [ ] kill logs
- [ ] command logs
- [ ] mod calls
- [ ] show banned players
- [ ] show current vehicles
- [ ] join logs
